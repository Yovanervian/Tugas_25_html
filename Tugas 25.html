Mengenal Semantic Elements Di HTML5

  • Articel

    • Ketika ia ingin membuat layout untuk menampilkan blok artikel yang sudah ada di website kita, tag article bisa kita gunakan dengan Elements header dan footer. karna tag article memiliki type blok artinya Elements ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.

  • Aside

    • Elements ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menembahkan aside dibagian bawah postingan untuk menjelaskan identitas penulis artikel. Karna aside memiliki type block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
